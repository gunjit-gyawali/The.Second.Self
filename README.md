# The.Second.Self

## About the Game

In **The.Second.Self**, you control **two characters** with different abilities. The catch? You can only use one at a time. To make it through the levels, you’ll have to **switch between them**, think creatively, and figure out how to get past each obstacle.

## How to Play

- **Move:** Use **W, A, D** to move your current character.  
- **Switch Characters:** Press **E** to switch between the two characters.  
- **Restart Level:** Press **R** if you get stuck or want to try a different approach.  

**Tip:** Some puzzles can only be solved if you use the right character at the right time.

## Features

- Two playable characters
- Challenging puzzles that require switching between characters.  
- Easy-to-learn controls for smooth gameplay.  

## Why You’ll Enjoy It
